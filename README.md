# Ely Prism Launcher

## Project Description
Ely Prism Launcher is a user-friendly launcher designed for easy management and launching of Minecraft instances. It provides features like mod support, profile management, and an intuitive interface.

## Setup Instructions
1. **Clone the repository:**  
   `git clone https://github.com/lewandowskif560-sketch/ely-prism-launcher.git`

2. **Navigate to the project directory:**  
   `cd ely-prism-launcher`

3. **Install dependencies:**  
   Use a package manager to install the required dependencies (if applicable). 
   
   For example, if using Node.js:
   `npm install`

4. **Run the application:**  
   Execute the launcher using the command:  
   `npm start`  
   or the relevant command for your setup.

5. **Configure your instances:**  
   Follow the instructions within the launcher to add Minecraft instances.

6. **Enjoy!**  
   Launch your favorite Minecraft version and start playing!
